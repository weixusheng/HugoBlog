解决方式

更换渲染引擎

1
2

多行公式需要使用两个”$$"

1
2
3
4

$$\left \lbrace\begin{matrix}
    h_\Theta(X)>0.5, set h_\Theta(X) = 1\\

\end{matrix} \right.$$

$$\left \lbrace\begin{matrix} h_\Theta(X)>0.5, set h_\Theta(X) = 1\ h_\Theta(X)<0.5, set h_\Theta(X) = 0 \end{matrix} \right.$$
