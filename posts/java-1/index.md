## 命名规范

Classname: 使用帕斯卡命名法:

每个单词都要写``JavaName`

Methods: 使用驼峰命名法

除了第一个单词以外,每个单词的首字母都要大写`javaName`

```java
public class Main{
    public void main(){
        ...code
    }
}
```

## 函数参数

`parameter`是指函数定义中参数

`argument`指的是函数调用时的实际参数。

parameter=形参(formal parameter)

argument=实参(actual parameter)。
