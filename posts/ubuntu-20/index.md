# 2.插件推荐
1. Applications Menu
   topbar会显示一个快捷"应用程序"按钮
2. Clipboard Indicator
   复制粘贴的历史记录
3. Dash to Dock
   智能显示dock
4. Hide Activities Button
   隐藏topbar上的鸡肋active
5. OpenWeather
   天气插件,显示在topbar上
6. Recent Items
   打开文件的历史记录
7. TopIcons Plus
   将正在运行的程序图标移至topbar
8. User Themes
   可以改变shell主题的插件
9. Add Username to Top Panel
   在topbar中增加用户名

10. Bing Wallpaper Changer
   每日更新bing壁纸

11. Bluetooth quick connect
   蓝牙快速连接

12. Coverflow Alt-Tab
   魔幻任务切换界面 control+tab

13. Desktop Icons
   在任务界面增加icon

14. Drop Down Terminal
   terminal快捷键召唤

15. Poweroff Button on Topbar
   在topbar增加关机键
